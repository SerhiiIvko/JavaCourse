Тема: Функции агрегирования. Объединения.
Разработать систему журнала абонентского обслуживания оператора сотовой связи.
База должна позволять хранить данные
•	Данные о сотрудниках технической поддержки
•	Данные об обращениях абонентов
o	Дата обращения
o	Номер телефона абонента
o	ФИО абонента
o	Причина обращения
o	Решение проблемы
o	Дата решения проблемы (либо время, затраченное на решение проблемы)
Запросы:
Реализовать следующие запросы:
•	Выбрать всех абонентов, обратившихся за определенный период времени
•	Выбрать всех сотрудников, обслуживших более N абонентов
•	Выбрать всех абонентов, обратившихся с одинаковой проблемой
•	Выбрать всех абонентов, проблема которых не была решена в течении X времени
•	Выбрать среднее время обслуживания абонентов за месяц
•	Выбрать суммарное время обслуживания абонентов за месяц
•	Выбрать сотрудников и количество абонентов, которых они обслужили за месяц, отсортировать данные по убыванию
