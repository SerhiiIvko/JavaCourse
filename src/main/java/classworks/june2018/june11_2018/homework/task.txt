Тема: Функции агрегирования. Объединения.
Разработать систему учета успеваемости студентов ВУЗа
База должна позволять хранить данные:
•	Данные о сотрудниках ВУЗа
o	ФИО
o	Должность
o	Дата рождения
o	Контактные данные
o	Дополнительная информация
•	Данные о студентах ВУЗа
o	ФИО
o	Группа
o	Курс
o	Дата рождения
o	Контактные данные
o	Дополнительная информация
•	Данные о посещаемости и успеваемости студентов
•	Данные о читаемых предметах
•	Данные о занятости аудиторий

Запросы:
Реализовать следующие запросы:
•	Выбрать всех студентов, у которых была лекция у преподавателя X
•	Выбрать всех преподавателей, у которых сегодня пары в аудитории X
•	Выбрать всех студентов, которые пропустили более N предметов за X период отсортировать по убыванию.
•	Выбрать все аудитории, которые свободны в определенную дату
•	Выбрать всех студентов, у которых есть задолженности по определенному предмету.
