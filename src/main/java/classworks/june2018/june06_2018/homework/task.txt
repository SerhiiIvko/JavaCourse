Разработать базу данных для хранения информации, таблицы в базе должны быть приведены к 3й нормальной форме.
Обеспечить целостность данных.
Разработать систему учета товара в магазине.
База должна содержать в себе
•	Данные о товаре для продажи
•	Данные о продавцах
•	Данные о продажах
•	Данные о возвратах
•	Данные о браке
•	Информацию о перемещениях товара (допустим в другой магазин)
Запросы:
Реализовать следующие запросы:
•	Выбрать все товары, проданные за прошлый месяц
•	Выбрать всех продавцов, продавших более N товаров за прошлую неделю
•	Выбрать все бракованные вещи и отсортировать по цене
•	Выбрать весь товар, который был перемещен из одного магазина в другой
•	Выбрать всех продавцов, продавших N товаров за X дней ценой не менее Z
